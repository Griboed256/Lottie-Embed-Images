**Lottie Embed Images** — это мощный инструмент с графическим интерфейсом (GUI) для моушн-дизайнеров и frontend-разработчиков. Программа автоматически упаковывает Lottie-анимацию (экспорт из Bodymovin) и PNG-секвенцию в **один независимый (self-contained) JSON-файл**.

Вам больше не нужно передавать разработчикам архив с папкой `images/`. Все кадры сжимаются, кодируются в Base64 и встраиваются прямо в код анимации.

## ✨ Главные возможности
* **Всё в одном файле.** Скрипт переводит кадры в формат Base64 и вшивает их внутрь JSON. Итоговый файл сразу готов к имплементации на сайт или в приложение.
* **Умное WebP-сжатие.** Алгоритм "на лету" конвертирует тяжелые PNG-кадры в формат WebP (Quality или Lossless), кардинально уменьшая итоговый вес JSON-файла.
* **Автоподбор под лимит (Smart Limit).** Нужно уложиться строго в 3 MB? Просто задайте лимит ползунком! Фоновый алгоритм сам просчитает сотни вариантов и подберет идеальное качество WebP.
* **Умная пакетная обработка (Batch).** Поддержка Drag & Drop. Перетащите в окно целую папку с десятками проектов — программа сама найдет все валидные JSON-файлы, отбросит мусор и выстроит их в очередь.
* **Встроенная галерея предпросмотра.** После рендера программа поднимает локальный сервер и открывает браузер, где анимации отрисовываются через официальный `lottie-web` движок в 60 FPS.
* **Автокоррекция багов Bodymovin.** Скрипт автоматически находит и исправляет известную ошибку отсутствующего параметра `op` (Out Point) на последнем кадре sequence-слоев.
* **Современный Flat UI (Zero State).** Интерфейс написан на чистом Tkinter, но переработан до неузнаваемости. Интерактивные кастомные ползунки, цветовая индикация качества сжатия, защита от зависаний интерфейса и плавная отзывчивость.
* 🔄 **OTA Обновления.** Встроенная проверка релизов на GitHub. Если выйдет новая версия, программа сама предложит её скачать.

## 📥 Установка и запуск

Программа распространяется в виде готового `.exe` файла, не требующего установки Python и дополнительных библиотек.

1. Перейдите в раздел [Releases](https://github.com/Griboed256/Lottie-Embed-Images/releases).
2. Скачайте последнюю версию
   * `Lottie_Embed_Images.exe` для пользователей Windows
   * `Lottie_Embed_Images.dmg` для пользователей MacOS.
4. Запустите файл (программа работает портативно).

### Внимание пользователям macOS:
* Для запуска приложения нужно будет один раз нажать правой кнопкой мыши на приложение после установки из dmg и выбрать _«Открыть», либо зайти в Настройки -> Конфиденциальность -> Все равно открыть._
* Если при первом запуске система пишет, что файл поврежден, **откройте Терминал** и выполните команду:
`xattr -cr /Applications/LottieEmbedImages.app`

#### Это связано с тем, что приложение не имеет платной подписи Apple Developer.


## 📖 Инструкция по применению

**Подготовка в After Effects:**
1. Подготовьте композицию и экспортируйте PNG-секвенцию.
2. При экспорте через Bodymovin: **Выключите** `Include in JSON` и **Включите** `Copy to assets folder`.
3. На выходе у вас должен быть `.json` файл и папка `images/` рядом с ним.

**Работа в программе:**
1. Перетащите ваш `.json` файл (или папку с проектами) в окно программы.
2. Выберите формат сжатия (рекомендуется `WebP` для сильного снижения веса).
3. Настройте качество ползунком (программа сразу покажет примерный вес файла) или включите автоматическое ограничение размера.
4. Нажмите **«Запустить»**. 
5. Заберите готовый файл с суффиксом `_result.json`. Папка с картинками больше не нужна!
